# السراج للتجهيزات الصناعية | Al-Seraj Industrial Equipment 🏢🔥

الموقع الرسمي لشركة **السراج للتجهيزات الصناعية**، الوكيل المعتمد لشركة **Asenware** العالمية في الجمهورية اليمنية. تم تطوير هذا الموقع ليكون منصة تفاعلية ثنائية اللغة تدعم أحدث معايير الأمان الصناعي.

[معاينة الموقع المباشر 🌐](https://abualkarar.github.io/seraj-website/)

---

## 🚀 التحديثات العالمية الكبرى | Major Global Updates

- **دعم اللغة الإنجليزية (Multi-language Support)**: تبديل سلس بين العربية والإنجليزية لجذب العملاء المحليين والدوليين.
- **نظام التحقق من الأصالة (Authenticity Checker)**: أداة ذكية تتيح للعملاء التأكد من أن منتجاتهم أصلية عبر الرقم التسلسلي.
- **حاسبة الأنظمة التقديرية (System Estimator)**: أداة تفاعلية لتقدير احتياجات المنشآت من أنظمة الحماية.
- **مركز التحميلات (Download Center)**: قسم مخصص لتحميل الكتالوجات الفنية وشهادات الاعتماد الدولية (PDF).
- **تصميم متجاوب عالمي**: واجهة مستخدم متطورة تدعم اتجاهات النصوص (RTL & LTR) بشكل تلقائي.
- **شاشة تحميل واحتفالية بصرية**: تجربة دخول فخمة تعكس هوية الشركة.

## ✨ التحديثات الجديدة (2024) | New Updates

### 🎯 تحسينات الهيكل والأداء
- **فصل الملفات**: تم فصل CSS و JavaScript إلى ملفات منفصلة (`css/style.css` و `js/main.js`) لتحسين الصيانة والأداء
- **تحسين الأداء**: تقليل حجم الملفات وتحسين وقت التحميل
- **معالجة أفضل للأخطاء**: إضافة فحوصات للعناصر قبل التعديل عليها

### 📄 محتوى جديد
- **قسم الخدمات (Services)**: عرض شامل للخدمات المقدمة:
  - أنظمة الحريق والإنذار المبكر
  - عقود الصيانة الدورية
  - التدريب الفني والاستشارات
- **قسم الاتصال المحسّن (Contact)**: معلومات التواصل والموقع الجغرافي مع أيقونات
- **تحسين الترجمات**: إضافة ترجمات شاملة لجميع العناصر الجديدة

### 🎨 تحسينات التصميم
- **تأثيرات حركية محسّنة**: استخدام مكتبة AOS لتأثيرات الحركة السلسة
- **تحسين التفاعلية**: إضافة تأثيرات hover محسّنة على جميع العناصر
- **تحسين الاستجابة**: تحسين التصميم على الأجهزة المحمولة والشاشات الصغيرة
- **نظام الألوان المحسّن**: تحسين التباين والوضوح في الوضع الليلي والنهاري

### 🔧 تحسينات تقنية
- **تنظيم أفضل للكود**: هيكل منطقي وسهل الصيانة
- **دعم أفضل للمتصفحات**: توافق محسّن مع جميع المتصفحات الحديثة
- **تحسين SEO**: إضافة كلمات مفتاحية وتحسين البيانات الوصفية

## 🚀 كيفية الاستخدام | How to Use

### التشغيل المحلي
1. انسخ الملفات إلى مجلد محلي
2. افتح ملف `index.html` في متصفح الويب
3. أو استخدم خادم محلي:
```bash
# باستخدام Python 3
python -m http.server 8000

# ثم افتح http://localhost:8000
```

### النشر على GitHub Pages
1. ادفع الملفات إلى مستودع GitHub
2. اذهب إلى إعدادات المستودع
3. فعّل GitHub Pages من فرع `main`
4. سيكون الموقع متاحاً على `https://username.github.io/seraj-website`
